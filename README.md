## 📁 How to download the Project 
```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/ecommerce-nature.git

    # Enter directory
    $ cd ecommerce-nature

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
